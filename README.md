# go_fetch
CLA Course Project

This repository is a complete ROS 2 workspace for the "Go Fetch!" demo on B3V0.
It is intended to run on a shared robot environment, which already has all required system dependencies installed.

## Clone

git clone --recursive git@github.com:andonbreitenfeld/go_fetch_ws.git
cd go_fetch_ws

## Build

colcon build
source install/setup.bash
